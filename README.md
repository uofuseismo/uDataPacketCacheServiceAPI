# About

This repository is the API for the [uDataPacketCacheService](https://github.com/uofuseismo/uDataPacketCacheService) - an in-memory data packet cache.  This serves packets to near-real-time packets with low latency.  For data requests requiring data older than approximately five minutes it is recommended to use the [uWaveServer](https://github.com/uofuseismo/uWaveServer).
